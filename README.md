# GameSwap

Plateforme d'échange et de vente de jeux de société entre particuliers.

## Technologies

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Installation

```sh
# Cloner le repository
git clone <YOUR_GIT_URL>

# Naviguer vers le dossier du projet
cd gameswap

# Installer les dépendances
npm i

# Lancer le serveur de développement
npm run dev
```
